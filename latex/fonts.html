字体 ​

字体类型 ​

以黑板粗体为例，格式是这样的：$\mathbb{AB}$ ，显示为 AB\mathbb{AB}AB 。

黑板粗体（blackboard bold） ​

用 \mathbb 或 \Bbb ，仅用于大写字母，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\Bbb{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ

粗体（boldface） ​

用 \mathbf ，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathbf{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz\mathbf{abcdefghijklmnopqrstuvwxyz}abcdefghijklmnopqrstuvwxyz

对于基于表达式的字符，使用 \boldsymbol 代替， 如 α\boldsymbol{\alpha}α 。

斜体（italics） ​

用 \mathit ，这也是公式中默认的字体，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathit{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz\mathit{abcdefghijklmnopqrstuvwxyz}abcdefghijklmnopqrstuvwxyz

加粗斜体（boldfaced italics） ​

用 \pmb ，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\pmb{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz\pmb{abcdefghijklmnopqrstuvwxyz}abcdefghijklmnopqrstuvwxyz

打字机字体（typewriter） ​

用 \mathtt ，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathtt{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz\mathtt{abcdefghijklmnopqrstuvwxyz}abcdefghijklmnopqrstuvwxyz

罗马字体（roman） ​

用 \mathrm ，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathrm{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz\mathrm{abcdefghijklmnopqrstuvwxyz}abcdefghijklmnopqrstuvwxyz

无衬线字体（sans-serif） ​

用 \mathsf ，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathsf{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz\mathsf{abcdefghijklmnopqrstuvwxyz}abcdefghijklmnopqrstuvwxyz

书法字体（calligraphic） ​

用 \mathcal ，仅用于大写字母，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathcal{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ

手写体（script） ​

用 \mathscr ，仅用于大写字母，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathscr{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ

德国尖角字体（旧德文风格）（Fraktur） ​

用 \mathfrak ，如

ABCDEFGHIJKLMNOPQRSTUVWXYZ\mathfrak{ABCDEFGHIJKLMNOPQRSTUVWXYZ}ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz\mathfrak{abcdefghijklmnopqrstuvwxyz}abcdefghijklmnopqrstuvwxyz

字体大小 ​

以 large 为例，格式是这样的：$\large{LaTeX}$ ，显示为 LaTeX\large{LaTeX}LaTeX 。

字体颜色 ​

用 \color{色值}{字符} ，其中色值可以用 red 、green 等常用颜色，也可以用十六进制 rgb 形式，如

\color{red}{F=ma} 显示为 F=ma\color{red} F=maF=ma ，\color{#228B22}{F=ma} 显示为 F=ma\color{#228B22}{F=ma}F=ma 。
